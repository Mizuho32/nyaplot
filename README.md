# Ecolirb

the Ruby front-end library of Ecoli.js

## Installation

Add this line to your application's Gemfile:

    gem 'ecolirb'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ecolirb

## Usage
   ```ruby
   require 'ecolirb'
   Ecolirb.init_iruby
   bar = Ecolirb.Bar.new([5,14,2,9])
   bar.show
   ```
